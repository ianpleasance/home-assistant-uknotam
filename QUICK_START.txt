🚀 QUICK INSTALL

   1. Copy folder to Home Assistant:
      cp -r notam/ /config/custom_components/

   2. Add to configuration.yaml:
      notam:
        coordinates:
          - latitude: 51.4700    # London area
            longitude: -0.4543
            range_nm: 50
          - latitude: 53.3537    # Manchester area
            longitude: -2.2750
            range_nm: 30
        refresh_interval: 60

   3. Restart Home Assistant:
      ha core restart

   4. Verify in Developer Tools → States:
      Look for sensor.notam_* entities

📋 EXAMPLE CONFIGURATIONS

   Single Area (backward compatible):
   notam:
     coordinates:
       latitude: 51.4700
       longitude: -0.4543
       range_nm: 50

   Multiple Areas (NEW!):
   notam:
     coordinates:
       - latitude: 51.4700
         longitude: -0.4543
         range_nm: 50
       - latitude: 53.3537
         longitude: -2.2750
         range_nm: 30

   With Airfields:
   notam:
     airfields:
       - "EGLL"
       - "EGKK"
     coordinates:
       - latitude: 51.4700
         longitude: -0.4543
         range_nm: 50
